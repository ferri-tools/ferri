# Documentation

This directory contains all documentation for the Ferri project.

-   **user_guide/**: Documentation for end-users.
    -   **examples/**: Example flows.
-   **project/**: Documents related to the development of the project.
    -   **adrs/**: Architectural Decision Records.
    -   **development_logs/**: Logs from development sessions.
    -   **product_planning/**: Product planning documents.
-   **archive/**: Archived documents.
    -   **dubious_docs/**: Old and potentially outdated documents.
    -   **proposed_flows/**: Proposed flow examples.
